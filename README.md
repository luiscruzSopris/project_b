# Example Project

A simple Python project for practicing Git workflows.

## Description

This repository contains a minimal Python program used as a sandbox for learning and experimenting with Git. It's a good starting point for trying out commits, branches, merges, rebases, and remotes without worrying about breaking anything important.

## Requirements

- Python 3.7 or higher

## Installation

Clone the repository:

```bash
git clone 
cd 
```

## Usage

Run the program with:

```bash
python main.py
```
